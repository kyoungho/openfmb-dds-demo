"dds.transport.shmem.builtin.received_message_count_max"
RTI Connext DDS API Reference » Pluggable Transports » Built-in Transport Plugins » Shared Memory Built-in Transport Properties

This does not guarantee that the Transport-Plugin will actually be able to buffer received_message_count_max messages of the maximum size set in "<TRANSPORT_PREFIX>.parent.message_size_max". The total number of bytes that can be buffered for a transport plug-in is actually controlled by receive_buffer_size.
