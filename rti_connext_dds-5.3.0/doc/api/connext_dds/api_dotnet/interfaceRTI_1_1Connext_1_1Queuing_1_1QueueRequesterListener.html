Detailed Description

Called when certain events occur in a RTI::Connext::Queuing::QueueRequester<TReq,TRep>.

A QueueRequester listener is a way to implement a callback that will be invoked when certain events happen. It is an optional parameter in RTI::Connext::Queuing::QueueRequesterParams<TReq,TRep>.
