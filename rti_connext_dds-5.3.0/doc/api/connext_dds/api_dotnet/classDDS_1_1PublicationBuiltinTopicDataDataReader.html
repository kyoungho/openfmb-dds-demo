Detailed Description

Instantiates DataReader < DDS::PublicationBuiltinTopicData > .

DDS::DataReader of topic DDS::PublicationBuiltinTopicDataTypeSupport::PUBLICATION_TOPIC_NAME used for accessing DDS::PublicationBuiltinTopicData of the remote DDS::DataWriter and the associated DDS::Publisher.
