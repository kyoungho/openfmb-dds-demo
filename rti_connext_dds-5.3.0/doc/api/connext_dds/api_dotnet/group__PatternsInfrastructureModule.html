Detailed Description

Infrastructure types for RTI Connext Messaging.

This section describes the sample types used for reading and writing. These types encapsulate data and related information:

  • RTI::Connext::Infrastructure::Sample<T> (used for reading)
  • RTI::Connext::Infrastructure::WriteSample<T> (used for writing)

There are different operations that provide loaned samples from the middleware. They return a RTI::Connext::Infrastructure::LoanedSamples<T>.
