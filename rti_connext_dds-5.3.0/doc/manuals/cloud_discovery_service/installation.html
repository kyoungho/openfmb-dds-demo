2. Installation¶

Cloud Discovery Service comes pre-installed as part of RTI Connext Professional Edition. If you have a different edition of RTI Connext you can still install it separately.

Contact support@rti.com for information on how to obtain a Cloud Discovery Service package for your platform. There are two ways to install Cloud Discovery Service: using RTI Launcher or the rtipkginstall command-line utility.

Note

Cloud Discovery Service is supported only in Connext version 5.3.0 and higher.

2.1. Installing RTI Cloud Discovery Service from RTI Launcher¶

To install Cloud Discovery Service from RTI Launcher:

  1. Start RTI Launcher from the Start menu or from the command line, by running: <NDDSHOME>/bin/rtilauncher.
  2. From the Utilities tab, click on RTI Package Installer.
  3. Use the + sign to add the .rtipkg file that you want to install.
  4. Click Install.

2.2. Installing RTI Cloud Discovery Service Using Command-Line¶

To install Cloud Discovery Service using the command-line, run:

$NDDSHOME/bin/rtipkginstall \
   /path/to/cloud/discovery/service/rtipkgfile.rtipkg
